# ImageJ-Macro-for-zebrafish-High-Content-Imager
Macro for batch analysis of zebrafish images taken on high content imager
These files are FIJI macro made to either crop or stitch images from high content imager and perform colocalization analysis, correct images and automatically generate data.
These were specifically created for the analysis of fluorescent and confocal images of zebrafish larvae xenotransplanted with cancer cells.
